#  Paola Deco & Eventos

Plataforma web para la gestión y visualización de un catálogo de servicios de decoración de eventos. Este proyecto es una aplicación **Full Stack (MERN)** desarrollada con propósitos educativos para dominar el flujo completo de desarrollo web moderno.

##  Tecnologías Utilizadas

### Frontend (Cliente)
* **React + Vite**: Para una interfaz rápida y reactiva.
* **Tailwind CSS**: Diseño moderno, responsivo y personalizado (tema de marca configurado).
* **React Router DOM**: Navegación SPA (Single Page Application).
* **Axios**: Consumo de API REST.

### Backend (Servidor)
* **Node.js + Express**: Servidor RESTful API.
* **MongoDB Atlas**: Base de datos NoSQL en la nube.
* **Mongoose**: Modelado de datos (Schemas) y conexión a DB.
* **Dotenv**: Gestión de variables de entorno.

---

##  Funcionalidades (Estado Actual)

*  **Conexión Cliente-Servidor-DB**: Flujo de datos completo funcionando.
*  **Catálogo Público**: Visualización dinámica de servicios traídos desde MongoDB.
*  **Gestión de Servicios**: Formulario para crear nuevos servicios (título, precio, categoría, descripción).
*  **Diseño Responsivo**: Adaptado a móviles y escritorio.
